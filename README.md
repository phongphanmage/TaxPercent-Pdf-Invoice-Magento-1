# Tax Percent Pdf Invoice - Magento 1

## Installation

1. Go to magento 1 root folder

2. Copy/Paste module folder

3. Go To admin panel - Stores - Configuration - Pdf Print-outs - Invoice / Enable show tax percent

4. That's it.
